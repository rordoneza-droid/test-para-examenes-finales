## 💡 Estrategia de Memorización (Lo importante)

### ¿Por qué las "últimas palabras"?

En cada pregunta, las últimas 1-2 palabras de la respuesta son **únicas**. No se repiten. Esto facilita memorizar porque:

- **Menos información para memorizar** (2 palabras vs. una frase completa)
- **Más distintiva** (cada respuesta termina diferente)
- **Más eficiente** (recordar el final → recordar todo)

**Ejemplo real:**
```
P: "¿Qué garantiza el Art. 191 CRE?"
Respuesta larga: "Garantizar el acceso a la justicia de personas 
                  en indefensión o con limitaciones económicas, 
                  sociales o culturales, de forma gratuita"

Clave a memorizar: "o culturales" (las últimas 2 palabras)

¿Por qué? Porque nadie más termina con "o culturales"
en sus respuestas. Es único, distintivo.
```

## 🎯 Técnica: Spaced Repetition (Repetición Espaciada)

Este es el método científicamente probado más efectivo:

**Cronograma de repetición:**
- **Día 1:** Primera exposición (hoy)
- **Día 2:** Repite (+1 día)
- **Día 4:** Repite (+2 días)
- **Día 10:** Repite (+6 días)
- **Día 30:** Repite (+20 días)

**Resultado:** Si lo recuerdas en todos los intervalos = **MEMORIZADO PERMANENTE**
